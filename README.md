# SeekingEval:TerminalBench-Compatible Task Generator

A self-improving deep research agent running 24×7, designed to continuously discover and generate tasks compatible with the [Harbor framework](https://github.com/harbor-framework/harbor).

This project does not build Harbor itself. Instead, it focuses on creating high-quality tasks and task collections that follow Harbor’s task format and can be used for evaluation, benchmarking, and training workflows.

## Vision

Benchmarks should not remain static.
They should be dynamic, self-evolving, and continuously improving to push models and agents forward.

By continuously discovering new task opportunities and refining existing ones, this project aims to help build more adaptive and challenging benchmark ecosystems on top of Harbor.
